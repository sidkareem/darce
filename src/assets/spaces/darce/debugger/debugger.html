Calc Group Parameters
Parameter ID Description Value
QUERY The standard Script parameter 'QUERY' has no effect and should be set to OFF
WRITE Whether to write back data to the database or run in simulation
CALC_GROUP_ID List out the calculation group IDs to run in sequence
OUTPUT_TEMP_DATA_Y_N Output temporarily queried and calculated data in the log or not
SHOW_SQL_Y_N Display SQL statements in the log or not
CHECK_LOCKS_Y_N Check work status locking when writing back data or not
CHECK_SECURITY_Y_N Check user security data access when writing back data or not
DARCE_DIRECT_WRITE_Y_N Whether to write data directly to the database or go through BPC posting engine
DARCE_VERSION Which installed DarCE version to run on
OUTPUT_CALC_CUBES Tell what calculation table(s) to output to the log
OUTPUT_MINI_CUBES Tell what data source (mini) table(s) to output to the log
OUTPUT_RESULTS Tell what result table(s) to output to the log
OUTPUT_WRITE_BACK_TBLS Tell what write back table(s) to output to the log
DB_PLATFORM Identify the database name on which the engine runes
DEBUG_Y_N Run the engine in Debugging mode
DROP_ORPHANED_TEMP_TABLES_Y_N Clean up any obsolute temporary table from previous runs
EXECUTE_SQL_Y_N Execute the generated SQL statements or not
KEEP_TEMP_TABLE_DATA_Y_N Keep temporary queried and calculated table for debugging use or not
LARGE_DATASET_Y_N HANA DB specific parameter to tell the engine to use columnar tables or not
LOG_LEVEL What level of logging to write
OUTPUT_SNAP_TO_LOG_Y_N Whether to output the snapshot and delta tables to the log
SNAPSHOT_PCT Set the SQL WHERE clause member threshold by percentage in the snapshot table
SNAPSHOT_THRESHOLD Set the SQL WHERE clause member threshold by count in the snapshot table
SUPPRESS_ROW_COUNTS_Y_N Whether to write the row counts to the log or not
